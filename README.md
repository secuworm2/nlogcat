# nlogcat

Android 및 iOS 기기의 로그를 Windows에서 실시간으로 확인하고 분석할 수 있는 로그 뷰어입니다.

---

## 시스템 요구사항

| 항목 | 요구사항 |
|---|---|
| **OS** | Windows 10 이상 |
| **Android** | Android SDK Platform Tools (`adb`) 가 PATH에 등록되어 있어야 합니다 |
| **iOS** | iTunes (Apple Mobile Device Support) 가 설치되어 있어야 합니다 |

> ADB가 설치되어 있지 않다면 [Android SDK Platform Tools](https://developer.android.com/tools/releases/platform-tools) 페이지에서 다운로드 후 PATH에 추가하세요.

> iTunes가 설치되어 있지 않다면 [Apple 공식 사이트](https://www.apple.com/itunes/) 또는 Microsoft Store에서 설치하세요.

---

## 설치 및 실행

1. [Releases](../../releases) 페이지에서 최신 버전의 `nlogcat.exe` 또는 `.msi` 설치 파일을 다운로드합니다.
2. `.msi` 파일은 설치 마법사를 통해 설치합니다. `.exe` 파일은 별도 설치 없이 바로 실행 가능합니다.
3. 기기를 USB로 연결합니다.
   - **Android**: USB 디버깅을 활성화합니다.
   - **iOS**: 기기에서 "이 컴퓨터를 신뢰하시겠습니까?" 팝업이 표시되면 **신뢰**를 탭합니다.
4. `nlogcat.exe`를 실행합니다.

> Android USB 디버깅 활성화: 설정 → 개발자 옵션 → USB 디버깅 켜기  
> 개발자 옵션이 보이지 않는다면: 설정 → 휴대폰 정보 → 빌드 번호를 7번 탭합니다.

## 사용 방법

### 기기 연결 및 로그 보기

1. 앱 실행 후 상단 드롭다운에서 연결된 기기를 선택합니다.
   - Android 기기는 `[Android] 모델명 (시리얼)` 형식으로 표시됩니다.
   - iOS 기기는 `[iOS] 기기명` 형식으로 표시됩니다.
2. 선택 즉시 로그 스트리밍이 시작됩니다.
3. 새 로그가 들어오면 자동으로 아래로 스크롤됩니다. 중간 로그를 보려면 스크롤을 위로 올리면 자동 스크롤이 일시 중지됩니다.

### 로그 필터링

| 방법 | 설명 |
|---|---|
| 레벨 버튼 | 클릭으로 해당 레벨 표시/숨김. Android: V D I W E F / iOS: N D I W E F |
| 검색창 | 입력한 키워드를 포함한 로그만 표시. 일치하는 부분이 강조됩니다 |
| `[Aa]` 버튼 | 검색 대소문자 구분 토글 |

#### 로그 레벨 의미

**Android**

| 레벨 | 이름 | 설명 |
|---|---|---|
| V | Verbose | 최상세 로그 |
| D | Debug | 디버그 정보 |
| I | Info | 일반 정보 |
| W | Warning | 경고 |
| E | Error | 오류 |
| F | Fatal | 치명적 오류 |

**iOS**

| 레벨 | 이름 | 설명 |
|---|---|---|
| N | Notice | 일반 알림 (Default 포함) |
| D | Debug | 디버그 정보 |
| I | Info | 일반 정보 |
| W | Warning | 경고 |
| E | Error | 오류 |
| F | Fault | 치명적 오류 |

### 로그 상세 보기

로그 행을 **더블클릭**하면 전체 메시지를 팝업으로 확인할 수 있습니다.

### 스트림 제어

| 버튼 | 설명 |
|---|---|
| ⏸ 정지 | 로그 수신을 일시 중지합니다. 기기에서는 계속 로그가 발생합니다. |
| ▶ 재개 | 정지 이후 로그 수신을 다시 시작합니다. |
| 🗑 초기화 | 현재 화면의 로그를 모두 지웁니다. |
| 💾 저장 | 현재 표시된 로그를 `.txt` 파일로 저장합니다. |

## 설정

우측 상단 **설정** 버튼을 클릭하면 설정 창을 열 수 있습니다.

| 항목 | 기본값 | 설명 |
|---|---|---|
| 테마 | 다크 | 다크 / 라이트 선택 |
| 폰트 크기 | 12 | 8 ~ 24 범위 조절 |
| 폰트 | 기본 | 시스템에 설치된 고정폭 폰트 선택 |
| 최대 버퍼 줄 수 | 50,000 | 초과 시 오래된 로그부터 삭제 |
| adb 경로 | 자동 | 비워두면 PATH에서 자동 탐색 |

설정은 변경 즉시 자동 저장됩니다.

## 자주 묻는 질문

**Q. Android 기기가 드롭다운에 나타나지 않아요.**  
A. USB 디버깅이 활성화되어 있는지 확인하세요. 기기 화면에 "USB 디버깅을 허용하시겠습니까?" 팝업이 떠 있다면 허용을 눌러주세요. 그래도 안 된다면 USB 케이블을 다시 연결해보세요.

**Q. iOS 기기가 드롭다운에 나타나지 않아요.**  
A. iTunes(Apple Mobile Device Support)가 설치되어 있는지 확인하세요. 기기를 처음 연결할 경우 iPhone 화면에 "이 컴퓨터를 신뢰하시겠습니까?" 팝업이 표시되면 **신뢰**를 탭해야 합니다.

**Q. "iTunes가 설치되어 있지 않습니다" 경고가 나와요.**  
A. [Apple 공식 사이트](https://www.apple.com/itunes/) 또는 Microsoft Store에서 iTunes를 설치한 후 앱을 재시작하세요.

**Q. iOS 연결 후 로그가 수신되지 않고 오류가 표시돼요.**  
A. iPhone 화면에서 신뢰 팝업을 아직 수락하지 않은 경우입니다. USB를 재연결한 뒤 팝업이 표시되면 **신뢰**를 탭하세요.

**Q. "ADB를 찾을 수 없습니다" 오류가 나와요.**  
A. ADB가 설치되지 않았거나 PATH에 등록되지 않은 경우입니다. 설정 창에서 `adb.exe`의 전체 경로를 직접 입력하거나, [Android SDK Platform Tools](https://developer.android.com/tools/releases/platform-tools)를 설치 후 PATH에 추가하세요.

**Q. 로그가 너무 빠르게 올라가요.**  
A. 레벨 버튼으로 불필요한 레벨을 숨기거나, 검색창에 키워드를 입력해 원하는 로그만 표시하세요. ⏸ 정지 버튼으로 스트림을 멈춘 뒤 살펴볼 수도 있습니다.

**Q. 한글 로그가 깨져 보여요.**  
A. 설정에서 폰트를 시스템에 설치된 다른 폰트로 변경해보세요.
